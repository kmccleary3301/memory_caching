# memory-caching 0.1.0

Date: 2026-03-06

## Summary

Initial public PyPI release of `memory-caching`.

PyPI:

- `https://pypi.org/project/memory-caching/`

## Included in 0.1.0

- Stable runtime package surface:
  - `memory_caching.MCConfig`
  - `memory_caching.MemoryCachingLayer`
  - `memory_caching.SegmentCache`
  - `memory_caching.LinearMemoryBackend`
  - `memory_caching.DLABackend`
  - `memory_caching.TitansBackend`
  - `memory_caching.SWLABackend`
- MIT license metadata and root `LICENSE`
- Clean wheel/sdist/dev install smoke
- Repo engineering gate
- Scientific gate with model-backed benchmark artifacts and non-smoke targets

## Important boundaries

- This is a public package release, not a claim of full paper parity.
- Missing paper baselines such as `Log-Linear++` still block literal table-level
  parity claims.
- Research tooling in this repository remains broader than the stable package
  API exported at the top level.
